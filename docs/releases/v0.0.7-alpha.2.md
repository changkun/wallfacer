# v0.0.7-alpha.2

## 🔧 Follow-up patch release

- **Release date:** 2026-04-14
- **Previous release:** `v0.0.7-alpha.1` (2026-04-13)
- **Delta:** 3 commits

A small follow-up to `v0.0.7-alpha.1` to fix a release-build regression and deflake two tests. No feature changes.

---

## 🐛 Fixes

- **Sandbox image tag now tracks `latere-ai/images`, not wallfacer's own version.** Release binaries previously pulled `ghcr.io/latere-ai/sandbox-claude:v<wallfacer-version>`, which frequently did not exist because sandbox images are published from a separate repository on an independent cadence. The binary now embeds the latest `latere-ai/images` release tag at build time (resolved in both `Makefile` and the release workflows) and falls back to a runtime API lookup, then `:latest`, when no tag is embedded.

## 🧪 Test stability

- Deflaked `TestMetricsGauge_SubscriberCount` in `internal/handler`.
- Deflaked `TestBoardCacheHit` in `internal/runner` on slow runners.

---

## 📦 Upgrade notes

No migration required. Just install the new binary — it will start pulling the correct sandbox image tag on first task run.
