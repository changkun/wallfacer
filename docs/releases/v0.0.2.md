# v0.0.2

## What's Changed since v0.0.1

### New Features

- **Mobile-responsive layout** — Kanban board now adapts to mobile screens with horizontal scrolling, snap-to-column navigation, and bottom-sheet modals

### Bug Fixes

- **Container git worktrees** — Mount repo `.git` directory into sandbox containers so git worktree operations work correctly inside the sandbox
- **Snapshot extraction** — Fix rsync skipping same-size files when extracting non-git snapshots
- **Container monitor** — Fix `escapeHTML is not defined` ReferenceError in the container monitor UI
- **Header sizing** — Fix header size regression in styles.css
- **Fonts & styling** — Restore Google Fonts (Inter, Instrument Serif) and Tailwind CSS preflight reset that were lost during offline bundling
